
# 정산서 상세 페이지

## 배경
- 정산서(`Settlement`) 상세 페이지를 정산계약 상세와 동일 패턴으로 맞춤
- 메타바 3행 (정산서/계약서/거래처), 섹션 구조/순서/수수료 레이아웃 통일

## 변경 사항
- 메타바 3 row × 3 pair — 아코디언으로 1행만 유지 가능
- 섹션 순서: 정산서 기본 정보 → 정산 증빙 → 정산 지급 → 수수료 → 첨부파일 → 시스템 → 계약 정보(스냅샷) → 공급자 거래처
- 수수료 정보 섹션 패턴 일치: 수수료 계산 방법 + 서비스 포인트/대장페이 per-row VAT 제외/포함 토글
- `.detail-section` 자동 아코디언 적용 (기본 열림)
