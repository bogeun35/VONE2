
# 정산서 관리 페이지 신규 + 그룹 헤더 6종

## 배경
- 정산서(`Settlement`) 리스트 페이지 신설
- 컬럼이 45+ 로 많아 시각적 구분 필요

## 변경 사항
- AG Grid 2단 헤더 (group header)
  - 정산서 / 계약 / 거래처 / 매출 / 매입 / 이체 데이터 6 그룹
- 금액 유무 필터 3종(매입/매출/지급) — `전체 / 있음 / 없음`
- 증빙 상태 옵션 — 전체 / 증빙 생성 필요 / 실패 / 완료 / 승인 대기 / 국세청 전송
- 푸터 6버튼 — 정산서 생성 / 계좌 검증 / 이체 생성 / 일괄 잠금 / 잠금 해제 / 일괄 수정
