# 이체관리

## 개요
정산서 기반으로 생성된 이체 건을 관리하는 페이지입니다.
이체 상태(대기/요청됨/성공/실패)별 현황을 확인하고, 상태를 변경할 수 있습니다.

---

## 필터 조건

### 조건 설정
| 필터 | 타입 | 옵션 |
|------|------|------|
| 이체 상태 | 셀렉트박스 | 전체, 대기, 요청됨, 성공, 실패 |
| 거래처 구분 | 셀렉트박스 | 전체, 직영, 가맹, 기타 |
| 출금 계좌 | 셀렉트박스 | 전체, 계좌별 목록 |

### 기간 설정
- **간편 검색**: 전체 / 오늘 / 이번 달 / 지난 달 (버튼 그룹)
- **지급예정일**: 날짜 범위
- **생성일자**: 날짜 범위
- **이체요청일**: 날짜 범위

### 키워드 검색
이체관리 IDX, 정산서 IDX, 거래처, 계약서명, 예금주명, 계좌번호, 제휴점

---

## 요약 영역

상단에 이체 상태별 합계를 표시합니다.

| 상태 | 표시 정보 |
|------|-----------|
| 대기 | 금액 합계 / 건수 |
| 요청됨 | 금액 합계 / 건수 |
| 성공 | 금액 합계 / 건수 (파란색 강조) |
| 실패 | 금액 합계 / 건수 (빨간색 강조) |

---

## 상태 변경 규칙

- **이체 성공** 상태: `실패 처리` 버튼 표시
- **이체 실패** 상태: `성공 처리` + `이체 확인` 버튼 표시
- **대기** 상태: `이체 요청` 버튼 표시
- **요청됨** 상태: 변경 불가
