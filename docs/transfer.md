# 이체관리 기획문서

## v1.1.0 변경사항 요약

### 필터 영역 변경
- **이체 상태**: 드롭다운 → 라디오버튼 UI 변경
  - [전체][이체 대기][이체 승인][실행 요청][이체 실행][이체 성공][이체 실패]
  - 디폴트: `이체 대기`
- **거래처 구분**: 드롭다운 → 라디오버튼 UI 변경
  - [전체][법인][개인]
- **간편 검색**: `전체` 버튼 제거, 지급예정일/생성일자/이체요청일 모든 날짜 필터에 간편검색 추가

### 요약 영역 변경
- `승인됨` 항목 추가 (보라색 강조)

### 테이블 컬럼 추가

| 추가 컬럼 | 설명 |
|-----------|------|
| 지급예정일 | 이체 지급 예정 날짜 |
| 대표자명 | 사업자 대표자 이름 |
| 사업자명 | 사업자 상호명 |
| 사업자 구분 | 개인/법인 뱃지 표시 |
| 예금주 검증(대표자) | 예금주 vs 대표자명 일치율 백분위 표시 |
| 예금주 검증(사업자) | 예금주 vs 사업자명 일치율 백분위 표시 |
| 예금주 검증(종합) | 사업자구분에 따라 개인→대표자 / 법인→사업자 기준 표시 |

> 일치율 60% 미만 시 **빨간 배경** 강조

### 성공 처리 모달 신규

처리 전 모달이 표시됩니다:
- **거래일시**: 범위 입력 (디폴트: 오늘)
- **금액 범위**: 범위 입력 (디폴트: 해당 이체의 금액)
- **입출금 구분**: `입금` 고정
- `조회` 버튼으로 통장 거래내역에서 매칭 건 검색

### 버그 수정

- `[수정]` 요약 필드에 의해 AG Grid 프레임 높이가 활성화된 크롬 영역을 초과하여 약 100vh만큼 크게 나타나 스크롤이 지속적으로 필요한 상황 → **그리드 높이를 반응형(flex)으로 변경하여 뷰포트에 맞게 자동 조절**

### 기타 UI 개선
- 필터 영역 아코디언 접기/펼치기 처리
- 하단 플로팅 글로벌 푸터 추가
- 페이지네이션 단위: 1,000 / 2,000 / 5,000 / 10,000 / 20,000건
- AG Grid 필터 아이콘: 호버 시에만 표시 (컬럼 텍스트 짤림 방지)
- 체크박스 중앙 정렬 보정
