# 정보갱신 버튼 추가

## 배경

정산서가 갱신되었을 때, 이미 생성된 이체 건의 사업자 정보 / 계좌 / 금액이 최신 정산서와 싱크되지 않는 문제. 기존에는 해당 이체 건을 삭제 후 재생성해야 했음.

## 요구사항

- `이체 대기` / `이체 승인` 상태 행에 `정보갱신` 버튼 노출
- 고유 정산서 단위이므로 모달 없이 **즉시 실행** (filled 파란 버튼)
- 클릭 시 최신 정산서 정보를 해당 이체 건에 덮어쓰기

## 비고

- 이전 명칭 `정산서 갱신` → `정보갱신` 으로 리네이밍 (PROD-22012 결정)
- 이체 실행 / 성공 / 실패 상태에서는 노출하지 않음
