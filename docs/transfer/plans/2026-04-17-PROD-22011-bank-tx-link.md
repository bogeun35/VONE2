# 통장거래 IDX 컬럼 추가

## 배경

이체 성공 건이 실제로 어떤 통장 거래내역과 연결되었는지 그리드에서 즉시 확인이 불가능했음. 성공처리 모달에서 연결 후 별도 화면에서 재조회해야 했음.

## 요구사항

- 그리드에 `통장거래 IDX` 컬럼 추가 (기존 `입금 IDX` 명칭 변경)
- 이체 성공 상태일 때만 값 표시 (그 외는 빈 셀)
- LinkRenderer 스타일 적용 (파란색 링크 텍스트)
- 우측 정렬, 숫자 필터 사용

## 변경 사항

- 컬럼 폭: 70px
- headerClass: `header-right`
- field: `depositIdx` (sample: `563570 + seq % 20`)
