# 이체관리 조건 필터 + 컬럼 Pin 고정

> 이 파일은 **본문(설명)** 만 담습니다. 제목/상태/기획자/태그/이슈번호 등 메타데이터는 **기획문서 매니저**(`docs/plans-index.json` + localStorage overlay) 에서 관리합니다.

## 배경
- 이체관리 리스트에서 조건 검색/기간/키워드 3행 필터 구조 확립
- 빈번 조회 컬럼 양쪽 고정 필요

## 변경 사항
- 조건 검색: 계약서 타입 (멀티체크), 이체 상태 (라디오), 거래처 구분, 출금 계좌 셀렉트
- 컬럼 Pin: 이체관리 IDX (좌), 이체예정금액 · 검증(%) (우)
- 계약구분 컬럼 추가 (정산계약 기반 파생)
