# 정산 계약 관리 정책문서

정산 계약 관리는 거래처와 체결한 정산 계약(식권대장/복지대장/퀵대장/단체선물대장/기타)을 조회·생성·관리하는 페이지입니다.

> 본 문서는 정산 계약 관리에 **특화된 정책만** 정리합니다.
> 공통 UI / 버튼 색상 규칙 / 탭·LNB 같은 기반 정책은 별도 공통 정책문서를 참고하세요.

---

## 1. 페이지 개요

- URL 앵커: `#정산계약관리`
- 탭 ID: `settle-contract-list`
- 본문: 조건 검색 + 계약 목록 테이블 + 푸터(신규 생성)
- 상세보기 / 정산서 보기 / 신규 생성은 **새 탭**으로 오픈 (`detailOf: 'settle-contract-list'`)

## 2. 조건 검색 필드

| 구분 | 항목 | 타입 | 비고 |
| --- | --- | --- | --- |
| 조건 | 계약서 타입 | 멀티셀렉트 버튼 | 식권/복지/퀵/단체선물/기타 |
| 조건 | 태그 | 서치+드롭다운+칩 | 기타비용·로봇·이용료·기기수수료 (확장 예정) |
| 조건 | 증빙 수취 유형 | 멀티셀렉트 | 공급자발행·시스템발행·종이세금계산서 |
| 조건 | 증빙 상계 / 이체 상계 | 라디오 | 전체·상계·별도 |
| 조건 | 그룹 정산 여부 | 라디오 | 전체·그룹·개별 |
| 조건 | 계약 상태 | 라디오 | 전체·유효·만료 |
| 조건 | 폐업 여부 | 라디오 | 전체·정상·폐업 |
| 날짜 | 계약시작일 / 종료일 / 해지일 | date range | 오늘·이번 달·지난 달 프리셋 |
| 키워드 | 계약/거래처/사업자번호/제휴점/담당자 | 텍스트 | — |

## 3. 테이블 컬럼

- 기본 폭을 **타이트하게** 설정해 대부분 한 화면에 보이도록 함 (사용자 리사이즈 후 템플릿 저장 권장)
- IDX 계열은 폭 40~44 고정
- 우측 고정: 정산서 보기 / 상세보기

## 4. 상세보기 / 신규 생성

- 현재 **빈 탭 페이지**로 오픈 (추후 구현)
- 탭 ID 규칙
  - 상세: `settle-contract-detail-{idx}` (idx 별 개별 탭)
  - 신규: `settle-contract-new` (단일 탭)
  - 정산서 이동: `settle-list-by-contract-{idx}`
